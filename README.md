# AMBIOM BFF CSV Repository

This repository contains structured metadata CSV files for microscopy datasets, designed for use with the [BioFileFinder](https://bff.allencell.org/) resource.

## Datasets

### S-BIAD2133 - Neutrophil Microscopy Images

**Original Dataset:** [BioImage Archive S-BIAD2133](https://www.ebi.ac.uk/biostudies/bioimages/studies/S-BIAD2133)

This dataset contains microscopy images of human and mouse neutrophils, including both training data and test data with various anomaly types (air bubbles, artifacts, contamination, illumination issues, and z-shift problems).

### VesselExpress - Vessel Light-Sheet Microscopy with Segmentation Masks (in different organs)

**Original Dataset:** [Zenodo](https://zenodo.org/records/6025935)
**Publication:** [DOI](https://zenodo.org/records/6025935)

This dataset contains raw, segmented and skeletonized 3D light-sheet microscopic image volumes of blood vessels of different organs which were processed by VesselExpress.
